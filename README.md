
### Приложение аналог инстаграмма

## Возможности
* Регистрация пользователя
* Авторизация
* Добавление имя профиля и аватара
* Просмотр картинок других пользователей
* Добавление/удаление своих картинок
* Возможность ставить лайк

## Технологии
* React
+ React hook's
+ React router
* API
* HTTP
* БЭМ
* html, css, js
